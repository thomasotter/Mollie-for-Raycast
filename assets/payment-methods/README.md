# Payment Method Icons

This folder contains icons for different payment methods supported by Mollie.

## Required Icons

Place PNG images (recommended size: 32x32 or 64x64 pixels) for the following payment methods:

### Main Payment Methods
- `creditcard.png` - Credit Card / Mastercard / Visa
- `ideal.png` - iDEAL
- `bancontact.png` - Bancontact
- `banktransfer.png` - Bank Transfer / SEPA
- `paypal.png` - PayPal
- `applepay.png` - Apple Pay

### Buy Now, Pay Later
- `klarna.png` - Klarna (Pay Later / Slice It)

### Gift Cards & Vouchers
- `giftcard.png` - Gift Cards
- `voucher.png` - Vouchers

### European Payment Methods
- `sofort.png` - SOFORT Banking
- `eps.png` - EPS (Austria)
- `giropay.png` - Giropay (Germany)
- `belfius.png` - Belfius (Belgium)
- `kbc.png` - KBC/CBC (Belgium)
- `przelewy24.png` - Przelewy24 (Poland)
- `mybank.png` - MyBank (Italy)

## Icon Guidelines

- **Format:** PNG with transparency
- **Size:** 32x32, 48x48, or 64x64 pixels (Raycast automatically scales)
- **Style:** Simple, recognizable logos/icons
- **Background:** Transparent or white

## Where to Find Icons

You can get official payment method logos from:
- Mollie's brand assets: https://www.mollie.com/en/press
- Individual payment provider websites
- Icon libraries like Flaticon, Icons8 (check licensing)

## Fallback

If an icon is missing, the extension will fall back to Raycast's built-in BankNote icon.
